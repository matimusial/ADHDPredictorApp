Żeby dostać się do widoku admina wciśnij przycisk "Admin View" po lewej stronie okienka programu (tego po uruchomieniu programu).

USER VIEW - Przywraca do głównego okna
GAN for MRI - Okno z uczeniem modelu GAN do generowania zdjęć MRI
CNN for MRI - Okno z uczeniem modelu CNN dla zdjęć MRI
CNN for EEG - Okno z uczeniem modelu CNN dla fal EEG

=CNN for EEG=====================================================================================:
    - "Chose folder with data" - (przycisk) typ tekstowy,          domyślnie rf'{parent_dir}/EEG/TRAIN/TRAIN_DATA'
        Folder z danymi uczącymi musi mieć następującą strukturę:
        (wybrany folder)
                        |
                        | ADHD
                        |    |
                        |    [pliki grupy z ADHD .mat/.csv/.edf]
                        |
                        | CONTROL
                                |
                                [pliki grupy kontrolnej .mat/.csv/.edf]

    - "Epochs"          -   typ liczbowy,               domyślnie 20
    - "Batch size"      -   typ liczbowy,               domyślnie 32
    - "Learning rate"   -   typ zmiennoprzecinkowy,     domyślnie 0.001
    - "Electrodes"      -   typ liczbowy,               domyślnie 19
    - "Frame size"      -   typ liczbowy,               domyślnie 128
    - "Frequency"       -   typ liczbowy,               domyślnie 128

    - "Start"           -   (przycisk) Uruchamia proces uczenia modelu

    - [PLOT]            -   (graf) Wyświetla wartość Accuracy i Loss po uruchomieniu uczenia w czasie rzeczywistym
